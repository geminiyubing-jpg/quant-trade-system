## 1. 系统登录

## 3. 实时行情

### 3.1 访问路径

点击侧边栏 **市场数据** → **实时行情**，或直接访问 `/market/realtime`

### 3.2 订阅股票

#### 方式一：单个订阅
1. 在输入框中输入股票代码（如 `000001.SZ`）
2. 点击「订阅」按钮
3. 股票将添加到行情表格中

#### 方式二：批量导入
1. 点击「批量导入」按钮
2. 输入多个股票代码（每行一个）
```
000001.SZ
000002.SZ
600000.SH
600036.SH
```
3. 点击「确认导入」

### 3.3 行情数据说明

| 字段 | 说明 |
|------|------|
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 最新价 | 当前最新成交价 |
| 涨跌额 | 相比昨收的涨跌金额 |
| 涨跌幅 | 相比昨收的涨跌百分比 |
| 成交量 | 今日累计成交量（手） |
| 成交额 | 今日累计成交额（元） |
| 最高 | 今日最高价 |
| 最低 | 今日最低价 |
| 今开 | 今日开盘价 |
| 昨收 | 昨日收盘价 |

### 3.4 设置价格预警

1. 点击股票行的「预警」按钮
2. 设置预警条件：
   - **价格上涨到**: 设置目标价格
   - **价格下跌到**: 设置止损价格
   - **涨跌幅超过**: 设置百分比阈值
3. 选择通知方式（邮件/站内通知）
4. 点击「保存」

### 3.5 取消订阅

1. 选中要取消的股票
2. 点击「取消订阅」按钮
3. 确认操作

## 4. 策略管理

### 4.1 访问路径

点击侧边栏 **策略管理**，或直接访问 `/strategy`

### 4.2 创建新策略

1. 点击「新建策略」按钮
2. 填写策略基本信息：

```
┌─────────────────────────────────────────┐
│  新建策略                                │
├─────────────────────────────────────────┤
│  策略名称: [__________________]          │
│  策略类型: [动量策略      ▼]             │
│  描述:     [__________________]          │
│            [__________________]          │
├─────────────────────────────────────────┤
│  策略参数                                │
│  ├─ 回看周期: [20___] 天                 │
│  ├─ 阈值:     [0.05_]                    │
│  └─ 止损比例: [0.03_]                    │
├─────────────────────────────────────────┤
│         [取消]        [创建策略]         │
└─────────────────────────────────────────┘
```

### 4.3 策略类型说明

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 动量策略 | 追涨杀跌，捕捉趋势 | 趋势明显的市场 |
| 均值回归 | 低买高卖，回归均值 | 震荡市场 |
| 因子策略 | 基于因子选股 | 量化选股 |
| 套利策略 | 利用价差获利 | 统计套利 |

### 4.4 版本管理

#### 创建新版本
1. 选择一个策略
2. 点击「创建版本」
3. 修改参数
4. 保存为新版本

#### 激活版本
1. 在版本列表中选择要激活的版本
2. 点击「激活」按钮
3. 确认操作

#### 版本回滚
1. 选择要回滚到的历史版本
2. 点击「回滚」按钮
3. 确认回滚操作

### 4.5 策略配置

每个策略包含以下配置模块：

#### 资金配置
- 初始资金
- 单只股票最大仓位
- 最大持仓数量

#### 风险配置
- 止损比例
- 止盈比例
- 最大回撤限制

#### 执行配置
- 交易时段
- 下单方式（市价/限价）
- 滑点容忍度

### 4.6 审计日志

系统记录所有策略操作：
- 创建时间、操作人
- 参数修改记录
- 版本变更历史
- 回滚操作记录

## 5. 策略回测

### 5.1 访问路径

点击侧边栏 **策略回测**，或直接访问 `/backtest`

### 5.2 创建回测任务

1. 选择要回测的策略
2. 设置回测参数：

```
┌─────────────────────────────────────────┐
│  创建回测任务                            │
├─────────────────────────────────────────┤
│  策略:     [动量策略_v1.2  ▼]            │
│                                          │
│  开始日期: [2024-01-01] 📅               │
│  结束日期: [2024-12-31] 📅               │
│                                          │
│  初始资金: [¥1,000,000___]               │
│  手续费率: [0.0003______] (0.03%)        │
│  印花税:   [0.001_______] (0.1%)         │
│  滑点:     [0.001_______] (0.1%)         │
├─────────────────────────────────────────┤
│         [取消]        [开始回测]         │
└─────────────────────────────────────────┘
```

3. 点击「开始回测」
4. 等待回测完成

### 5.3 回测结果

#### 绩效指标

| 指标 | 说明 |
|------|------|
| 总收益率 | 整个回测期间的累计收益 |
| 年化收益率 | 折算的年度收益率 |
| 夏普比率 | 风险调整后收益 |
| 最大回撤 | 最大亏损幅度 |
| 胜率 | 盈利交易占比 |
| 盈亏比 | 平均盈利/平均亏损 |

#### 扩展指标

| 指标 | 说明 |
|------|------|
| Sortino 比率 | 下行风险调整后收益 |
| Calmar 比率 | 年化收益/最大回撤 |
| Alpha | 超额收益 |
| Beta | 市场敏感度 |
| VaR (95%) | 95%置信度下的最大损失 |
| CVaR (95%) | 极端情况下的平均损失 |

### 5.4 因子分析

因子分析展示策略中各因子的有效性：

| 因子 | IC | IC_IR | 因子收益 | 换手率 |
|------|------|------|----------|--------|
| momentum_20 | 0.045 | 2.1 | 1.2% | 35% |
| volume_ratio | 0.032 | 1.8 | 0.8% | 42% |

**指标说明**：
- **IC (Information Coefficient)**: 因子与收益的相关性，绝对值越大越好
- **IC_IR**: IC 的信息比率，衡量因子的稳定性
- **因子收益**: 因子带来的超额收益
- **换手率**: 因子导致的持仓变动频率

### 5.5 归因分析

归因分析拆解收益来源：

```
收益归因
├── 配置效应: +2.5%  (行业配置贡献)
├── 选股效应: +3.2%  (个股选择贡献)
└── 交互效应: +0.8%  (配置与选股的交互)
────────────────────
总超额收益: +6.5%
```

### 5.6 导出报告

1. 点击「导出报告」按钮
2. 选择导出格式（PDF/Excel）
3. 选择要包含的内容
4. 下载报告文件

## 6. 交易管理

### 6.1 访问路径

点击侧边栏 **交易管理**，或直接访问 `/trading`

### 6.2 交易模式

系统支持两种交易模式：

| 模式 | 说明 | 风险 |
|------|------|------|
| **模拟交易** | 使用虚拟资金，不涉及真实交易 | 无风险 |
| **实盘交易** | 连接真实券商账户，执行真实交易 | 有风险 |

#### 切换交易模式
1. 点击顶部「交易模式」选择器
2. 选择目标模式
3. 确认切换

> ⚠️ **警告**: 切换到实盘交易前，请确保：
> - 已充分测试策略
> - 了解相关风险
> - 设置了合理的风控参数

### 6.3 下单操作

#### 限价单
1. 输入股票代码
2. 选择买卖方向（买入/卖出）
3. 输入价格和数量
4. 点击「下单」

```
┌─────────────────────────────────────────┐
│  下单                                    │
├─────────────────────────────────────────┤
│  股票代码: [000001.SZ    ] 🔍           │
│  股票名称: 平安银行                       │
│  当前价格: ¥12.50                        │
│                                          │
│  ○ 买入    ○ 卖出                        │
│                                          │
│  订单类型: [限价单      ▼]               │
│  价格:     [¥12.48______]                │
│  数量:     [1000________] 手             │
│                                          │
│  预估金额: ¥12,480                       │
│  预估佣金: ¥3.74                         │
├─────────────────────────────────────────┤
│         [取消]        [确认下单]         │
└─────────────────────────────────────────┘
```

#### 市价单
1. 选择「市价单」类型
2. 输入数量
3. 系统以当前最优价格成交

### 6.4 订单管理

| 操作 | 说明 |
|------|------|
| 查看订单 | 查看所有订单及状态 |
| 撤单 | 取消未成交的订单 |
| 修改 | 修改未成交订单的价格/数量 |

### 6.5 成交记录

查看所有成交历史：
- 成交时间
- 股票代码/名称
- 买卖方向
- 成交价格/数量
- 佣金、印花税、过户费

### 6.6 交易日历

查看交易日历，了解：
- 交易日/休市日
- 交易时段
- 特殊安排（如节假日调整）

### 6.7 交易统计

查看每日交易统计：
- 订单数量
- 成交数量
- 成交金额
- 交易费用
- 当日盈亏

## 7. 投资组合

### 7.1 访问路径

点击侧边栏 **投资组合**，或直接访问 `/portfolio`

### 7.2 创建组合

1. 点击「新建组合」
2. 输入组合名称和描述
3. 选择基础货币（CNY/USD）

### 7.3 持仓管理

#### 查看持仓

| 股票 | 数量 | 成本价 | 现价 | 市值 | 盈亏 | 权重 |
|------|------|--------|------|------|------|------|
| 000001.SZ | 1000 | ¥12.00 | ¥12.50 | ¥12,500 | +¥500 | 15% |
| 600000.SH | 500 | ¥10.00 | ¥10.50 | ¥5,250 | +¥250 | 8% |

#### 调整权重

1. 点击「调整权重」
2. 设置目标权重
3. 系统计算需要买卖的数量
4. 确认调整

### 7.4 风险分析

#### 风险指标

| 指标 | 说明 | 当前值 |
|------|------|--------|
| VaR (95%) | 95%置信度下最大损失 | 2.5% |
| CVaR (95%) | 极端情况平均损失 | 3.5% |
| 集中度 | 持仓集中程度 | 32% |
| 分散化比率 | 分散化效果 | 0.85 |
| Beta | 相对市场敏感度 | 1.12 |
| 波动率 | 收益波动程度 | 18% |

#### 行业分布

查看持仓的行业分布饼图，避免过度集中。

### 7.5 组合优化

系统提供多种优化方法：

| 方法 | 说明 | 目标 |
|------|------|------|
| 均值方差 | 经典优化方法 | 给定风险下最大收益 |
| 风险平价 | 风险均衡分配 | 每个资产贡献相同风险 |
| 最大夏普 | 最大风险调整收益 | 夏普比率最大化 |
| 最小方差 | 最小化组合风险 | 波动率最小化 |

#### 执行优化

1. 选择优化方法
2. 设置约束条件：
   - 单只股票最大权重
   - 单只股票最小权重
   - 行业权重限制
3. 点击「优化」
4. 查看优化建议
5. 确认执行

## 10. 常见问题

### Q1: 如何修改密码？

1. 点击右上角头像
2. 选择「设置」
3. 点击「安全设置」
4. 点击「修改密码」
5. 输入旧密码和新密码
6. 保存

### Q2: 忘记密码怎么办？

1. 在登录页面点击「忘记密码」
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击链接设置新密码

### Q3: 如何添加自选股？

**方式一**: 在实时行情页面输入股票代码订阅

**方式二**: 在自选股管理页面批量导入

### Q4: 回测结果和实盘差异大？

可能原因：
1. **滑点设置**: 检查滑点参数是否合理
2. **交易成本**: 确认手续费、印花税设置
3. **流动性**: 回测假设可以即时成交，实盘可能无法成交
4. **市场变化**: 历史表现不代表未来

### Q5: 如何导出数据？

1. 在相应页面点击「导出」按钮
2. 选择导出格式（Excel/CSV/PDF）
3. 选择导出范围
4. 下载文件

### Q6: 系统支持哪些数据源？

- **AkShare**: 免费，实时性一般
- **Tushare**: 需积分，数据全面
- **Wind**: 付费，专业级
- **东方财富**: 免费，实时性好

### Q7: 如何联系技术支持？

- **邮件**: support@quanttrade.example.com
- **工单**: 在系统内提交工单
- **文档**: 查阅 [API 文档](./API.md)
