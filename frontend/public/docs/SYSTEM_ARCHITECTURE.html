🔐 认证流程图

1 用户登录 →
2 验证凭据 →
3 生成 JWT →
4 返回 Token →
5 存储本地

Token 有效期: access_token 30分钟 | refresh_token 7天

1 下单请求 →
2 风控检查 →
3 提交券商 →
4 执行成交 →
5 状态推送

订单状态: PENDING → SUBMITTED → FILLED/CANCELED
