## 🐛 常见问题

### Q1: 页面显示 "Something went wrong"
**解决**: 按 F12 查看控制台错误，检查是否缺少依赖

### Q2: 看不到模式切换器
**解决**:
1. 确认服务器已启动
2. 清除浏览器缓存（Ctrl+Shift+R）
3. 检查控制台是否有错误

### Q3: 点击切换器没有反应
**解决**:
1. 按 F12 查看控制台错误
2. 检查 TradingModeContext 是否正确导入

### Q4: TypeScript 错误
**解决**:
```bash
npm install --save-dev @types/node @types/react @types/react-dom
```
